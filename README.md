# Space Gadgets

The Best place to get rare tools and shop for your favorite gadgets

## Installation

```bash
$ git clone https://github.com/damigreen/space-gadgets.git
$ yarn
$ yarn start
```

* [**Redux**](redux.js.org/) - A Predictable State Container for JS Apps
